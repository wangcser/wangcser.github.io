COPY

信息社会是关系社会的一种抽象，给与人类更多想象力的同时，也带了一些麻烦。

Mr. George

这篇文章的名字原来叫做 ABOUT ，本来应该写点个人简介？创作理念？生活态度？或者放一张奇丑无比的作者照片？后来我把它改成了 COPY ，因为我想和大家简单聊聊 COPY 。

COPY 又名 ctrl + c ，是大家在IT领域最熟悉不过的功能了。正是由于计算机的的某些特性，以及网络的某些特性，使得我们可以自由地复制数据，以及借助数据编码的信息。COPY 推动了一个时代，或许也是开启下一个时代的钥匙。

COPY 是上一个乃至目前 IT 社会的基石，但也由于 COPY 的存在，人类社会最伟大的发明——价值和权利，在目前的互联网上似乎有点行不通。

整个互联网改变的需求已经蠢蠢欲动了，我们现在基于互联网所确定的价值，其实都被锁定在了由各个互联网公司确定的一个一个的域内，虽然这也是人类社会价值确定的一种映射，但互联网似乎并没有帮上什么忙。

价值是能够 COPY 的吗？

朋友们可以将这个博客定义为一个技术博客 ——一个分享博主01世界历险记的地方，但这不是我的定义。

我不喜欢去分享一些技术细节（教大家如何编码，配置环境，使用某个小功能），因为在过去的技术学习以及今后人生的大部分时间的历练中，我似乎能够摸索出一套处理这些技术细节的方法，而这种方法其实每个成功入门IT技术领域的人都是知道并且掌握的。对于这种大家都知道的内容，我就没有 COPY 的必要了。

我更希望从另一个角度去思考技术——生活。我认为生活是技术的来源与归宿，繁琐技术细节的背后，都有一个大家习以为常的生活的定义——来自于人类文明千年的自然积累。但是做这样的思考是一个难度和风险都极高的行为，我从我目前遇到过极少的类似理念的文章中总结出了两点必要条件：

  1. 在自身领域有数年乃至数十年的积累与实践经验；
  2. 对基础知识和领域知识有深刻的理解和感悟。

而这两点都是目前的我所欠缺的，也就是说，在我的文章里，可能有一些合理性，但由于笔者自身的水平有限，也存在一些不合理之处，希望朋友们能够辨证地阅读。

S-Wong, M-Li
